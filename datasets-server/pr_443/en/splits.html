Splits

A dataset generally contains multiple splits, a specific subset of a dataset like train and test. The dataset can also contain configurations, a sub-dataset of the larger dataset.

See the documentation of the ðŸ¤— Datasets library to read more in depth about the concepts.

To get the list of splits and configurations of a dataset:

https://datasets-server.huggingface.co/splits?dataset={dataset_name}
Currently, the API only returns rows of the "streamable" datasets. By loading a dataset in streaming mode, the first rows can be extracted without downloading the whole dataset.

For example, the duorc dataset has six splits and two configurations:

https://datasets-server.huggingface.co/splits?dataset=duorc
